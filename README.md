# Darts Counter (Convex Auth)

This app uses:

- **Next.js** for the frontend
- **Convex** for database + backend functions
- **Convex Auth** for authentication

It supports:

- `1v1` and `2v2` game formats
- turn-by-turn darts scoring (301/501)
- bust handling (score cannot go below 0)
- per-player and per-team persistent statistics

## 1) Install and run

```bash
npm install
npx convex dev
npm run dev
```

Keep `npx convex dev` running in one terminal and `npm run dev` in another.

## 2) Configure environment variables

Copy `.env.example` to `.env.local`:

```bash
cp .env.example .env.local
```

`NEXT_PUBLIC_CONVEX_URL` and `NEXT_PUBLIC_CONVEX_SITE_URL` are auto-generated by `npx convex dev`.
Convex Auth backend variables (`SITE_URL`, `JWT_PRIVATE_KEY`, `JWKS`) are configured by:

```bash
npx @convex-dev/auth --skip-git-check --web-server-url http://localhost:3000
```

## 4) Where logic lives

- Convex schema: `convex/schema.ts`
- Convex backend logic: `convex/games.ts`
- Convex auth setup: `convex/auth.ts`
- Convex auth provider config: `convex/auth.config.ts`
- Frontend UI: `app/page.tsx`

## 5) Notes

- Game rotation for `2v2` is: A1 -> B1 -> A2 -> B2
- Stats tracked for each player and each team:
  - games played / won
  - turns played
  - points scored
  - busts
  - highest turn
